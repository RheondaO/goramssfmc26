# VCU-SFMC Student Financial Gap Calculator (MVP)
This repository contains a Pre-Implementation MVP of a Student Financial Gap Calculator, designed for the Student Financial Management Center (SFMC) at VCU.

## Project Overview
This repository serves as a functional Minimum Viable Product (MVP) for a Student Financial "Gap" Calculator, developed specifically for the Student Financial Management Center (SFMC) at Virginia Commonwealth University.

In alignment with the Information Technology Developer role, this project demonstrates the full SDLC: from substantial discovery and database design to the implementation of data-driven dashboards. The tool is designed to provide students and advisors with immediate clarity on out-of-pocket costs, directly supporting VCU‚Äôs goals for student retention and persistence.
